CI 의 코어 파일을 보던중 =& 라는 코드를 보았다. 이게 뭐지 해서 찾아봤더니

값을 넣을때 참조한 값을 넣는게 아닌 진짜 그값을 넣는거 같다.

<!--미리보기-->

예를 보니 바로 이해가 됐다.

```php
$array = array('apple', 'orange', 'banana');

// Without &
foreach($array as $d)
{
    $d = 'fruit';
}

echo implode(', ', $array); // apple, orange, banana

// With &
foreach($array as &$d)
{
    $d = 'fruit';
}

echo implode(', ', $array); // fruit, fruit, fruit
```
